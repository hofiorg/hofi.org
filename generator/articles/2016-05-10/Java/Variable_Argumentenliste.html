Um Parameter für Methoden variabel zu gestalten wurden mit Java 5 variable Argumentenlisten eingeführt. Die 3 Punkte kennzeichnen eine Parameterliste. Diese ist im weiteren Verlauf wie ein Array zu behandeln.
